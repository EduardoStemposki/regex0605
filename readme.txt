Explicações  sobre as expressões regulares utilizadas:

O atributo pattern especifica um padrão que o número de telefone deve seguir. 

O placeholder fornece um exemplo de formato para o usuário. 

O required faz com que este campo seja obrigatório.

A função validarFormulario() é chamada quando o formulário é enviado. Ela pega os valores dos campos do formulário, valida-os e exibe uma mensagem de alerta se algo estiver incorreto.
